# Voice & Copy Guidelines

## Language
Spanish (Spain). All user-facing text in the app is in Spanish.

## Tone
- **Professional and trustworthy** — financial context demands confidence
- **Clear and precise** — no ambiguity with money-related information
- **Warm but formal** — accessible premium, not cold corporate
- **Aspirational** — wealth creation, strategic investment, exclusivity

## Address
- Use formal "usted" for general UI text
- Use "tu/tú" only if brand guidelines explicitly switch to informal (TBD after first review)

## UI Text Patterns
| Context | Pattern | Example |
|---------|---------|---------|
| Button (primary) | Verb + object | "Ver inversiones" |
| Button (secondary) | Softer verb | "Explorar proyectos" |
| Button (cancel) | "Cancelar" or "Volver" | "Volver" |
| Empty state | Friendly + CTA | "Aún no tienes inversiones. Explora nuestros proyectos." |
| Error | What happened + action | "No se pudieron cargar los datos. Inténtalo de nuevo." |
| Success | Confirmation | "Datos actualizados correctamente." |
| Loading | — | Use skeleton/shimmer, not text |
| Section header | Noun or short phrase | "Proyectos destacados", "Últimas noticias" |
| Tab labels | Short noun | "Inicio", "Marcas", "Buscar", "Inversiones", "Mi perfil" |

## Number Formatting
- Currency: `€1.234.567,89` (European format: dots for thousands, comma for decimals)
- Percentages: `12,5%` (comma for decimals)
- Dates: `27 mar. 2026` or `27 de marzo de 2026` (full)

## Forbidden
- No ALL CAPS (except acronyms: VIP, ROI)
- No exclamation marks in error messages
- No technical jargon (no "API", "timeout", "null" in user-facing text)
- No informal slang
- No "click" — use "pulsa" or "toca"
